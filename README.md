# Monday.com BI Agent

AI-powered Business Intelligence Agent for Monday.com board data, tracking Deals, Work Orders, and project analytics.

## Project Structure

```text
Skylarp-drones-Software-Building-/
├── backend/            # FastAPI + Python backend
│   ├── venv/           # Virtual environment
│   ├── requirements.txt
│   └── .gitignore
├── frontend/           # React (Vite) + Tailwind CSS frontend
│   ├── src/
│   ├── package.json
│   ├── tailwind.config.js
│   └── postcss.config.js
├── .env.example        # Environment variable placeholders
├── UPDATES.md          # Project update & progress log
└── README.md
```

## Quick Start

### Backend
```bash
cd backend
# Activate virtual environment
# Windows PowerShell:
.\venv\Scripts\Activate.ps1
# Install dependencies
pip install -r requirements.txt
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
